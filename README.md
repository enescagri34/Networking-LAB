# CCNA 200-301 Certification Lab & Study Notes

Bu depo, **Cisco Certified Network Associate (CCNA 200-301)** sınavına hazırlık sürecimde gerçekleştirdiğim tüm ağ laboratuvarlarını, konfigürasyon dosyalarını ve teknik çalışma notlarımı içermektedir.
Daha çıktı eklemeleri devam edecektir

## 🛠 Kullanılan Araçlar
* **Simülasyon:** Cisco Packet Tracer, GNS3, EVE-NG
* **Cihazlar:** Cisco Catalyst Switches (2960, 3650), Cisco ISR Routers (4000 series)
* **Dokümantasyon:** Cisco IOS CLI, Markdown

## 📚 Müfredat ve Uygulama Alanları

### 1. Network Fundamentals
- [ ] OSI & TCP/IP Modelleri
- [ ] IPv4 Subnetting & IPv6 Adresleme
- [ ] Kablolama ve Fiziksel Katman

### 2. Network Access (Layer 2)
- [ ] VLAN & Trunking (802.1Q)
- [ ] **Spanning Tree Protocol (STP/RSTP)**
- [ ] **EtherChannel (LACP/PAgP)**
- [ ] Cisco Discovery Protocol (CDP) & LLDP

### 3. IP Connectivity (Layer 3)
- [ ] Static Routing (Standard, Floating, Default)
- [ ] **OSPFv2 (Single-area & Multi-area)**
- [ ] Inter-VLAN Routing (Router-on-a-Stick)

### 4. IP Services
- [ ] DHCP & DNS Konfigürasyonu
- [ ] NAT (Static, Dynamic, PAT)
- [ ] SNMP, NTP ve Syslog

### 5. Security Fundamentals
- [ ] Port Security
- [ ] Access Control Lists (Standard & Extended ACL)
- [ ] SSH & VTY Hat Güvenliği
- [ ] DHCP Snooping & Dynamic ARP Inspection (DAI)

### 6. Automation and Programmability
- [ ] SDN ve Kontrolcü Temelleri
- [ ] JSON, XML ve YAML Veri Yapıları
- [ ] Restconf & Netconf
